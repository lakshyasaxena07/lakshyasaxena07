### Featured Projects

**[🌾 Fasal Sarthi](https://github.com/lakshyasaxena07/Fasal-Sarthi-app)** — an AI-powered crop advisory system for farmers
- Crop disease detection using EfficientNetB3 (98.5% published accuracy)
- Crop recommendation via a Stacking Classifier, fertilizer recommendation via Random Forest
- Deployed with Flask + React on Vercel
- Holds a Government of India copyright (2025)
- Subject of my first-author IEEE paper at IECS 2026

**[🚗 Safe Overtaking Assistance System](https://github.com/lakshyasaxena07/Overtaking_safety.git)** — a real-time ADAS module for safer highway overtaking
- YOLOPv2 for perception, SORT tracker + Kalman filtering for object tracking
- Time-to-Collision (TTC) engine for overtaking risk assessment
- TensorRT quantization for real-time inference on edge hardware

**[📄 Resume Builder Application](https://github.com/lakshyasaxena07/ResumeBuilderApplication)** — a web app for building and managing resumes

**😴 Driver Drowsiness Detection System** — real-time eye-state and fatigue monitoring
- EfficientNetB0 CNN for eye-state classification, LSTM for temporal patterns
- MediaPipe landmarks for facial tracking, with a live alert pipeline

**🏠 Bengaluru House Price Prediction** — regression modeling on real estate data
